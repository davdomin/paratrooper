# Paratrooper (Estilo años 90)

Un juego retro de paracaidistas inspirado en los clásicos de los años 90.

## Características
- Jugabilidad al estilo arcade de los 90
- Controles simples y adictivos
- Sistema de puntuación y vidas
- Efectos visuales nostálgicos
- Música y efectos de sonido retro

## Cómo jugar
1. Compila el juego: `npx tsc`
2. Inicia el servidor: `npx http-server -p 8080` (desde el directorio paratrooper)
3. Abre tu navegador en: http://127.0.0.1:8080

## Controles
- Tecla izquierda/derecha: Mover el avión
- Barra espaciadora: Lanzar paracaidistas
- Tecla R: Reiniciar juego

## Estructura del proyecto
- `game.ts`: Lógica principal del juego (TypeScript)
- `index.html`: Estructura HTML del juego
- `styles.css`: Estilos visuales
- `dist/`: Carpeta de salida compilada (JavaScript)

## Requisitos
- Node.js (para TypeScript y servidor HTTP)
- Navegador web moderno
